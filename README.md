This repository aggregates project code from Max's MSc at Victoria University of Wellington.

- `msc-1-mnist/` — TensorFlow code for a conditional generative transformer on MNIST.
- `msc-2-mnist/` — exploratory notebooks for conditional MNIST prediction, image transformers, and ViT-style experiments.
- `msc-3-pytorch/` — PyTorch rewrite covering masked-MNIST modeling and transformer experiments on BVH hand animation data.
- `msc-4-hands/` — TensorFlow, Blender, and BVH tooling for building hand-motion datasets and training early hand-animation models.
- `msc-5-hands/` — later hand-animation training code with more advanced model and configuration experiments over synthetic and real hand data.
- `msc-6-hands/` — experiment workspace built around the extracted `mx` framework for MNIST and BVH tasks.
- `mx/` — shared utility and training framework for datasets, embeddings, pipelines, visualization, and transformer models used by the later experiments.
- `thesis/` — git submodule linking to the thesis document.
